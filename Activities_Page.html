How Code Creates These Games

These games demonstrate how programming concepts come together to create interactive experiences:

Game Loop & Animation

Both games use a continuous loop that updates the game state and redraws the screen 60 times per second, creating smooth animation.

Collision Detection

Algorithms check if game elements collide with each other or boundaries, creating the core gameplay mechanics.

AI Behavior

TRON enemies use decision-making algorithms to navigate the grid and try to trap the player.

Data Structures

Arrays and objects store game state including player positions, game statistics, and high scores.
